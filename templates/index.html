API Documentation

Use our REST API to integrate video conversion into your applications

Endpoints:
Convert WebM to MP4
POST /api/convert

Content-Type: multipart/form-data

Request: 
curl -X POST \
  -F "file=@video.webm" \
  {{ request.url_root }}api/convert
Response: 
{
  "success": true,
  "file_id": "uuid-string",
  "download_url": "/api/download/uuid"
}
Download Converted File
GET /api/download/<file_id> 
curl -O {{ request.url_root }}api/download/your-file-id
Service Status
GET /api/status
Request: 
curl {{ request.url_root }}api/status
Response: 
{
  "status": "online",
  "ffmpeg_available": true,
  "max_file_size_mb": 500
}
API Limits:
  • Max file size: 500MB
  • Supported format: WebM only
  • Conversion timeout: 10 minutes
  • Files auto-deleted after download
JavaScript Example:
// Convert file using JavaScript
const formData = new FormData();
formData.append('file', fileInput.files[0]);

fetch('/api/convert', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => {
    if (data.success) {
        // Download converted file
        window.location = data.download_url;
    }
});
