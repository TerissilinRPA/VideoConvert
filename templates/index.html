Video Converter & Creator

Convert your WebM video files to MP4 format or create videos from CSV files with image URLs

วิธีใช้งาน (ภาษาไทย)
แปลงไฟล์ WebM → MP4
  • เลือกแท็บ “WebM to MP4” แล้วอัปโหลดไฟล์ .webm (เลือกได้หลายไฟล์)
  • ติ๊ก “Enable Bulk Conversion Mode” หากต้องการส่งหลายไฟล์เข้าคิว
  • กด “Convert to MP4” ระบบจะประมวลผลด้วย FFmpeg
  • เสร็จแล้วกด “Download MP4” เพื่อดาวน์โหลด
สร้างวิดีโอจาก CSV
  • เลือกแท็บ “CSV to Video” แล้วอัปโหลดไฟล์ CSV (โครงสร้างอ้างอิงจาก example.csv)
  • ตั้งค่า Duration/Voice/Subtitles/Font/Watermark ตามต้องการ
  • กด “Create Video from CSV” ระบบจะดาวน์โหลดรูปและเรนเดอร์วิดีโอ พร้อมเสียงบรรยาย (Gemini TTS)
  • ผลลัพธ์จะแสดงลิงก์ดาวน์โหลดวิดีโอของแต่ละสินค้า
เรนเดอร์จากแอสเซ็ต (Render From Assets)
  • เรียก API POST /api/render-video พร้อม scriptData.scenes, images และ audioSrc (รองรับ data URL หรือ URL)
  • เปิดซับไตเติลบนวิดีโอได้ด้วย options.showSubtitles=true
  • ดาวน์โหลดไฟล์ที่เรนเดอร์ผ่าน GET /api/download-render/<file_id>

API Documentation

Use our REST API to integrate video conversion into your applications

Endpoints:
Convert WebM to MP4
POST /api/convert

Content-Type: multipart/form-data

Request: 
curl -X POST \
  -F "file=@video.webm" \
  {{ request.url_root }}api/convert
Response: 
{
  "success": true,
  "file_id": "uuid-string",
  "download_url": "/api/download/uuid"
}
Create Video from CSV
POST /api/csv-to-video

Content-Type: multipart/form-data

Request: 
curl -X POST \
  -F "file=@example.csv" \
  -F "duration_per_scene=4.0" \
  -F "voice_name=Zephyr" \
  -F "gemini_api_key=$GEMINI_API_KEY" \
  -F "show_subtitles=true" \
  -F "font_style=Sarabun" \
  -F "font_size=60" \
  -F "watermark=@yourchannel" \
  {{ request.url_root }}api/csv-to-video
Response: 
{
  "success": true,
  "file_id": "uuid",
  "product_videos": [
    {
      "product_id": 0,
      "product_title": "...",
      "download_url": "/api/download-product-video-ffmpeg//.mp4"
    }
  ]
}
Parameters:
  • file (required): CSV with product data and image URLs
  • duration_per_scene (optional): seconds per scene (default 5)
  • voice_name (optional): Gemini TTS voice (default Zephyr)
  • gemini_api_key (optional): API key (falls back to env)
  • show_subtitles, font_style, font_size, watermark
Download Converted File
GET /api/download/ 
curl -O {{ request.url_root }}api/download/your-file-id
Download CSV Product Video
GET /api/download-product-video-ffmpeg// 
curl -O {{ request.url_root }}api/download-product-video-ffmpeg/FILE_ID/PRODUCT_TITLE.mp4
Bulk Convert WebM
POST /api/bulk-convert

Content-Type: multipart/form-data

curl -X POST \
  -F "files=@v1.webm" -F "files=@v2.webm" \
  {{ request.url_root }}api/bulk-convert
Queue Status
GET /api/queue-status 
curl {{ request.url_root }}api/queue-status
Render From Assets
POST /api/render-video

Content-Type: application/json

Request (excerpt): 
{
  "scriptData": {"scenes": [{"text":"Hello","startTime":0,"endTime":2}]},
  "images": ["data:image/png;base64,...."],
  "audioSrc": "data:audio/mpeg;base64,....",
  "options": {"width":1080,"height":1920,"fps":30,"showSubtitles":true}
}
Response: 
{
  "success": true,
  "file_id": "uuid",
  "download_url": "/api/download-render/uuid"
}
Download Rendered Video
GET /api/download-render/ 
curl -O {{ request.url_root }}api/download-render/your-file-id
Service Status
GET /api/status
Request: 
curl {{ request.url_root }}api/status
Response: 
{
  "status": "online",
  "ffmpeg_available": true,
  "max_file_size_mb": 500
}
API Limits:
  • Max file size: 500MB
  • Supported formats: WebM, CSV
  • Conversion timeout: 10 minutes
  • Files auto-deleted after download
JavaScript Examples:
// Convert WebM file
const fd = new FormData();
fd.append('file', webmFileInput.files[0]);
const r = await fetch('/api/convert', { method: 'POST', body: fd });
const j = await r.json(); if (j.success) window.location = j.download_url;

// CSV → video
const fdcsv = new FormData();
fdcsv.append('file', csvFileInput.files[0]);
fdcsv.append('duration_per_scene', '4');
fdcsv.append('voice_name', 'Zephyr');
fdcsv.append('show_subtitles', 'true');
const r2 = await fetch('/api/csv-to-video', { method: 'POST', body: fdcsv });
const j2 = await r2.json();
if (j2.success && j2.product_videos?.length) window.location = j2.product_videos[0].download_url;

// Render from assets
await fetch('/api/render-video', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ scriptData, images, audioSrc, options: { width:1080, height:1920, fps:30, showSubtitles:true } })
});
